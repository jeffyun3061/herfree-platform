## 2. 자동 테스트 (코드)

로컬에서 실행 후 결과만 붙여 넣기.

```bash
cd backend && ./gradlew test
cd ../frontend && npm run lint && npm run build
```

| 항목 | 결과 | 비고 |
|------|------|------|
| Backend `./gradlew test` | ☐ PASS ☐ FAIL | 테스트 N개 |
| Frontend `npm run lint` | ☐ PASS ☐ FAIL | |
| Frontend `npm run build` | ☐ PASS ☐ FAIL | |
| GitHub Actions CI (있다면) | ☐ green ☐ red | PR/커밋 링크: |

**캡처 보관:** 터미널 PASS 화면 1장, (선택) GitHub Actions green 1장
