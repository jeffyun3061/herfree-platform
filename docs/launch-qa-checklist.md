# Herfree 런칭 전 점검표 (30분)

> **기능별·갭 상세:** [`launch-qa-detailed.md`](./launch-qa-detailed.md) (80항+, 만들 것 목록 포함)  
> 로컬: `http://localhost:3000` · 관리자: `/admin`  
> 체크: ✅ 됨 / ❌ 안 됨 / ⏭ 건너뜀

---

## 준비 (5분)

- [ ] Docker MySQL 켜짐 (`docker compose -f docker-compose.local.yml up -d`)
- [ ] 백엔드 켜짐 (`backend` → `./gradlew bootRun` 또는 `run-local.ps1`)
- [ ] 프론트 켜짐 (`frontend` → `npm run dev`)
- [ ] 크롬 **시크릿 창** 1개 + **일반 창** 1개 준비

## 결과

- **❌ 0개** → 베타(지인 몇 명) 열어도 됨  
- **❌ 1~3개** → 그것만 고치고 다시 1번부터  
- **❌ 4개 이상** → 개발자에게 ❌ 번호만 전달
